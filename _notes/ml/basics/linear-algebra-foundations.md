Linear algebra underpins most ML models:

- Vectors represent features and parameters.
- Matrices represent datasets and linear transformations.
- Dot products measure alignment and similarity.
- Matrix multiplication composes transformations.

A practical rule: if you can track tensor shapes confidently, model debugging gets much easier.
